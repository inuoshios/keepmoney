## KEEP MONEY

Finance manager written in TypeScript, Express JS, Zod for schema Validation, and JWT for authentication

Hear are some of the endpoints available

- Create User
- Get User
- Create Budget
- Update Budget
- GetSingleBudget
- GetAllABudget
- DeleteBudget
- CreateTransaction
- Get all transactions
- GetSingleTransaction,
- UpdateTransaction
- DeleteTransaction
